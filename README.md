# AI 工单分诊与回复 Agent

这是一个可直接演示的轻量级 Agent 项目，用于把客服/运营工单自动分类、判断优先级、匹配处理建议，并生成回复草稿与日报。

## 解决的痛点

客服/运营团队每天会收到大量重复工单，人工判断问题类型、紧急程度和负责人耗时较长，且回复口径容易不一致，导致 SLA 延迟和重复沟通。

## 核心逻辑流

1. **输入工单**：读取 `data/tickets.json` 中的用户问题。
2. **分类 Agent**：判断工单类型，例如账号、支付、物流、技术问题。
3. **优先级 Agent**：根据关键词识别紧急程度，例如“无法登录”“付款失败”“投诉”等。
4. **知识库检索**：从内置 SOP 中匹配建议处理方案。
5. **回复生成 Agent**：生成给用户的回复草稿。
6. **校验与汇总**：检查是否缺少关键字段，并输出每日处理报告。

## 运行方式

```bash
python src/main.py
```

运行后会在 `outputs/report.md` 生成分诊结果和统计报告。
